Island Mesh Generator

Java program to randomly generate different types of island maps with features such as size and biome customizable by user input. Additionally, there is a pathfinder using Dijkstra's Algorithm and Apache CLI library to transform the island mesh to a corresponding graph and find the minimum distance between 2 points. Built with Apache Maven.

Piratern Kapern Game Simulator

Java program that simulates play of German card game Piraten Kapern (see rulebook). 2 virtual players compete with strategy and rules given by the user. Program logs the move of each player and outputs the final result.
